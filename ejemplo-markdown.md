## Reglas importantes:

1. **Títulos de sección**: Deben terminar con `:` (dos puntos)
2. **Listas**: Usar `*` o `-` seguido de un espacio
3. **Espaciado**: Dejar una línea en blanco entre secciones
4. **Consistencia**: Usar el mismo símbolo (* o -) en toda la lista
